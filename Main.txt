I have make Csharp project of hospital management system. Where the users are patient, doctor, receptionist, nurse and phlebotomist. Everyone can sign up, log in, log out, edit profile. doctor can see patient list, update condition and assign tests. Patient Can see doctor list, see test list, see test report and book doctor. Receptionist can update doctor list, admit patient, book tests. Nurse can see patient list, update condition. Phlebotomist can see booked test list, submit test report.
